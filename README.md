# Docker+Apache+Php
*Contenedor docker para entornos de desarrollo drupal en Biko*

**Contiene:**
- Apache (SSL)
- Php 5.6/7.0/7.1 (gd, curl, opcache, memcached, redis, xdebug)
- Drush
- Drush console
- Composer

**Usado en:**
[Entorno para Desarrollo Drupal en Biko](https://github.com/biko2/docker-compose-drupal)
